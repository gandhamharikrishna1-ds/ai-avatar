# 🤖 AI Avatar Voice Assistant Web App

## 📌 Project Overview

This project is a web-based AI Voice Assistant with an animated avatar.
Users can interact using voice or text, and the system responds with speech while animating an avatar.

---

## 🚀 Features

* 🎤 Voice input using Speech Recognition
* 💬 Text input support
* 🔊 Voice output using Text-to-Speech
* 🧑 Animated avatar (Idle & Talking states)
* ⚙️ AI agent logic (basic command understanding)
* 🌐 Web-based interface (runs in browser)

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**APIs Used:**

* Web Speech API (SpeechRecognition)
* SpeechSynthesis API

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/gandhamharikrishna1-ds/ai-avatar.git
```

### 2️⃣ Navigate to project folder

```bash
cd ai-avatar
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the server

```bash
node server.js
```

---

## 🌐 Run the Project

Open your browser and go to:

```plaintext
http://localhost:3000
```

## 🎤 How to Use

1. Click the **microphone button**
2. Speak a command (e.g., "calculate 5 + 3")
3. The system:

   * Converts speech to text
   * Processes the request
   * Responds with voice
   * Animates the avatar

---

## 🧠 Example Commands

* "Calculate 10 + 5"
* "What is the weather?"
* "Hello"
